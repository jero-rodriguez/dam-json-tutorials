Guía de Estudio: JSON

Formato de intercambio de datos

¿Qué es JSON?

JSON (JavaScript Object Notation) es un formato de texto sencillo para enviar y recibir datos. Es el estándar actual en la web por su facilidad de lectura.

Reglas principales:

  • Los datos van en pares de nombre : valor.
  • Los datos se separan por comas.
  • Las llaves { } definen objetos.
  • Los corchetes [ ] definen arrays (listas).

Tipos de Datos admitidos

  • Cadenas: "Hola Mundo" (siempre con comillas dobles).
  • Números: 25 o 450.75.
  • Booleanos: true o false.
  • Nulos: null.
  • Objetos y Arrays.

Conversión de XML a JSON

Según la chuleta técnica, para pasar de XML a JSON debemos seguir estas pautas:

  1. Elementos simples: Se convierten en "clave":"valor".
  2. Elementos complejos: Se convierten en objetos { }.
  3. Elementos repetidos: Se convierten en arrays [ ].

Ejemplo Práctico

Datos cargados del objeto:

                {
                    "instituto": "IES Juan Bosco",
                    "web": "www.iesjuanbosco.es",
                    "ciclos": ["ASIR", "DAW", "SMR"]
                }
