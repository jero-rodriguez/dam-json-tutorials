Guía de MongoDB

Documentación original: Descargar guia-mongodb.docx

Características Generales

  • Tipo: SGBD NoSQL orientado a documentos.
  • Almacenamiento: Cada registro se almacena en un documento JSON (JavaScript Object Notation).
  • Lenguaje: JavaScript es el lenguaje de consulta por defecto.
  • Interoperabilidad: Existen drivers para Java, PHP, C#, C++, Python y Ruby.

Comparativa: Relacional vs. NoSQL

Modelo Relacional NoSQL (MongoDB)
Registros (rígidos) Documentos (flexibles)
Tabla Colección

MongoDB utiliza esquemas dinámicos, lo que permite crear colecciones sin definir una estructura previa.

Ventajas Principales

  • Modelado flexible de datos.
  • Alto rendimiento en escritura y consultas.
  • Alta escalabilidad.
  • Gestión de clúster de base de datos.

Herramientas de Ecosistema

MongoDB Atlas

Servicio de base de datos en la nube disponible en AWS, Azure y Google Cloud.

MongoDB Compass

Programa cliente para gestionar bases de datos, realizar consultas y crear estructuras de forma gráfica.

Ejemplos de Consultas (Filtrado)

  • Géneros: {genres : ["Drama", "Fantasy"]}.
  • Duración: {runtime: {$gt: 120}} (mayor a 120 minutos).
  • Puntuación: {"metacritic": {$gt: 80}}.
